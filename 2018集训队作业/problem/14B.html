画家小P

题面描述

小P最近迷上了画画，他常常欣赏名画来接受艺术熏陶。渐渐地，他摸索出了一些人们评价一幅画好坏的规律。画布上不同地方颜色的不同搭配，往往会使这 幅画展现出不同的样子。譬如，如果较大区域染上同一种颜色，会使得整幅画看起来略显单调。他还发现，人们往往将一幅画的美观程度定义为每个区块颜色的异或 和。

形式化地，我们将一张画布抽象成 n 个点 m 条无向边的简单无向图（无重边和自环），点标号为 1, 2, ..., n。设图中第 i 个点被染上的颜色为 coli，则一幅画的美丽值为 col1 ⊕ col2 ⊕ col3 ⊕ ... ⊕ coln （ ⊕ 表示异或和）。

一幅画是不单调的，当且仅当对于所有边 (u, v)，满足 colu ≠ colv。

现在小P得到了一张崭新的画布，对于每个点 i，他可选的颜色 coli 有限，必须是 0, 1, 2, ..., limiti 中的一个整数（limiti 为输入给定的数）。他想知道通过这张画布能画出多少幅不单调的美丽值为 C 的画。两种染色方案是不同的当且仅当存在至少一个点 u，使得 u 在两张图中的颜色 colu 不同。

由于答案可能较大，请输出答案对 998244353 取模的结果。

输入格式

第一行三个用空格分隔的整数 n, m, C，分别表示图的点数、边数和美丽值。

第二行 n 个用空格分隔的整数，分别表示 limit1, limit2, limit3, ..., limitn。

第 3 到第 m + 2 行每行两个用空格分隔的整数 u, v 表示一条无向边。

保证输入的图是无重边和自环的简单无向图。

输出格式

一行 1 个整数，表示答案对 998244353 取模的结果。

样例

样例输入一

3 2 3
1 3 2
1 2
3 1

样例输出一

4

样例一解释

有如下四种染色方案合法：

col1 = 0, col2 = 1, col3 = 2

col1 = 0, col2 = 2, col3 = 1

col1 = 1, col2 = 0, col3 = 2

col1 = 1, col2 = 2, col3 = 0

以下方案不合法：

col1 = 0, col2 = 3, col3 = 0 不满足 col1 ≠ col3。

样例输入二

5 7 711
724 449 696 581 118
5 2
3 4
5 4
1 3
5 3
1 4
2 4

样例输出二

62051347

样例二备注

注意输出答案对 998244353 取模后的结果。
