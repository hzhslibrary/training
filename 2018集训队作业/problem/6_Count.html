时间限制：2s　　　空间限制：512MB

题目描述

 给定01串s=s0s1...s|s|−1以及正整数N=∏ni=1piai。求有多少正整数k∈[1,N]满足

∀i∈[0,|s|),gcd(k+i,N)=1当且仅当si=1

 答案对109+7取模。

输入格式

 第一行输入只包含′0′,′1′的字符串s。

 第二行输入一个正整数n。

 接下来n行，每行输入两个整数pi和ai，保证pi为质数且互不相同。

输出格式

 输出1行，表示合法的正整数的数量。

样例1

input

01
2
3 2
5 1

output

15

样例2

input

1011
1
3 1000000000

output

411979884

限制与约定

 对于10%的测试点，N≤107；

 对于另外10%的测试点，n=1;

 对于另外20%的测试点，|s|≤15,pi≤50;

 对于100%的测试点，1≤|s|≤25,1≤n≤105,1≤pi,ai≤109。
