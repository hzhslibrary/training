多边形

题目描述

平面上有一片无限大森林，在每个整点 (x,y) (x,y≥0) 处都有一棵树，这棵树上有 Fx+y 个果实，其中 Fk 满足这样的递推式：

Fk=a⋅Fk−1+b⋅Fk−2+c⋅Fk−3+d⋅Fk−4

到了收获的季节，你要采摘一个范围内的果实，具体来说，你会采摘一个给定的 N 个顶点的多边形边界或内部的所有树上的所有果实，问你一共收获了多少个果实，请输出答案对 109+9 取模得到的结果。

输入格式

从标准输入读入数据。

输入数据的第一行包含一个整数 T，代表测试数据的组数，接下来是 T 组数据。

每组数据的第一行包含九个整数 N、a、b、c、d、F0、F1、F2 和 F3。

接下来 N 行，每行包含两个整数 xi 和 yi，代表多边形的一个顶点，顶点按照逆时针顺序给出。

输出格式

输出到标准输出。

对于每组数据，输出一行，包含一个整数，代表问题的答案。

样例输入

2
4 1 2 3 4 0 1 2 3
1 1
2 1
2 2
1 2
4 1 2 3 4 0 1 2 3
1 1
3 1
3 3
1 3

样例输出

18
153

样例解释

两组数据中的递推公式相同，为下面的公式：

Fk={kFk−1+2⋅Fk−2+3⋅Fk−3+4⋅Fk−4,0≤k≤3,k>3

因此 F4=10、F5=26、F6=63。

在第一组数据中，你会收获在 (1,1)、(1,2)、(2,1) 和 (2,2) 四个位置的果实，共有 F1+F2+F3+F4=18 个。

在第二组数据中，你会收获在 (1,1)、(1,2)、(1,3)、(2,1)、(2,2)、(2,3)、(3,1)、(3,2) 和 (3,3) 九个位置的果实，共有 F2+F3+F4+F3+F4+F5+F4+F5+F6=153 个。

评分方式

本题采用子任务捆绑测试。一个子任务可能包含多个测试点，只有通过一个子任务中的全部测试点，才能使该子任务得满分，否则该子任务得 0 分。

子任务

  • 1≤T≤2000；

  • 3≤N≤16；

  • 0≤a,b,c,d,F0,F1,F2,F3<109+9；

  • xi,yi≥0；

  • 多边形为非退化的简单多边形。

子任务分值特殊性质xi,yi各组数据中 N 之和
115无≤100≤6000
2101≤10000
315无
452≤109≤600
5133
6141
713无
815≤6000

特殊性质 1：多边形为凸多边形且不存在平行于 y 轴的边；

特殊性质 2：N=4，存在 0<u<v<w 使得四个顶点依次为 (0,v)、(u,v−u)、(u,w−u) 和 (0,w)；

特殊性质 3：N=3，存在 0<u<v<w 使得三个顶点依次为 (0,v)、(u,v−u) 和 (0,w)。
